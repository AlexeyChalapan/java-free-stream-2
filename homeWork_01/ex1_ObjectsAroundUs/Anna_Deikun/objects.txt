Объекты


1) Объект - телефон	
	Состояние:
		- вкл. выкл.
		- заряд батареи
		- картинка на экране
	Поведение:
		- включить - при включении устройство становится доступным для использования
		- выключить - при выключении телефон не приоден для использования 
		
		
2) Объект - носки
	Состояние:
		- процент износа
		- уровень загрязнения
	Поведение: 
		- изнашивается - носок становится тоньше и меньше согревает ногу
		- загрязняется - становится непригодным для носки
		- стирается - становится чистым и пригодным для носки
		
		
3) Объект - микроволновкаикроволновка
	Состояние:
		- вкл. выкл.
		- уровень мощности
		- таймер времени
	Поведение:
		- включить - при включении в микроволновке подогревается пища
		- выключить - микроволновка никем не используется и ничего не греет
		
		
4) Объект - колонка
	Состояние:
		- вкл. выкл.
		- уровень звука
	Поведение:
		- увеличение звука - музыка воспроизводится громче и увеличивается диапазон слышимости
		- уменьшение звука - музыка становится тише и никому не мешает
		
		
5) Объект - цветок
	Состояние:
		- свежий/несвежий
		- количество листиков
	Поведение:
		- завять - бутон опускается вниз, цветок увядает
		- распуститься - бутоны раскрывается, цветок живет и издает невероятный аромат
		
		
6) Объект - женская сумка
	Состояние:
		- уровень новизны
		- процент наполнения
	Поведение:
		- заполняется - в сумке много вещей, она становится тяжелой
		- освобождается - сумка выглядит миниатюрной, с ней легко ходить
		
		
7) Объект - дверь
	Состояние:
		- открыта/закрыта
		- есть/нет
	Поведение:
		- открывается - в помещение можно попасть
		- закрывается - вход в помещение заблокирован (вне зоны доступа)
		
		
8) Объект - вода
	Состояние:
		- температура
		- агрегатное состояние
	Поведение:
		- нагревается - становится теплой, можно принимать ванну
		- замерзает - становится твердой, можно кататья на коньках
		
		
9) Объект - ногти
	Состояние:
		- форма
		- длина
		- уровень загрязнения
	Поведение:
		- уменьшаются - становятся короткими, удобнее готовить
		- растут - становятся длинными и мешают
		
		
10) Объект - рана
	Состояние:
		- размер
		- глубина
	Поведение:
		- появляется - болит и кровоточит
		- пропадает - чешется и заживает
